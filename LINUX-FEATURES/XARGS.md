# XARGS

Команда `xargs` в Unix/Linux используется для преобразования стандартного ввода в аргументы команды. Она позволяет выполнять команду с большим количеством аргументов, чем может быть передано через командную строку.

Примеры использования команды `xargs`:

1. Вывести количество строк в каждом файле в текущей директории:
```
ls | xargs wc -l
```
2. Удалить все пустые файлы в текущей директории:
```
find . -type f -empty | xargs rm
```
3. Сжать все файлы с расширением `.txt` в текущей директории:
```
find . -name "*.txt" | xargs gzip
```
4. Вывести содержимое всех файлов с расширением `.log` в текущей директории:
```
find . -name "*.log" | xargs cat
```
5. Создать резервную копию всех файлов в текущей директории с помощью команды `cpio`:
```
find . -print | xargs cpio -ov > backup.cpio
```
В этих примерах команда `xargs` используется для преобразования вывода команд `ls`, `find` и других в аргументы команд `wc`, `rm`, `gzip`, `cat` и `cpio`.
